#### 21. ~~解决第20问题~~  <sub>2017-11-10</sub>

> * 通过调试存储过程发现p2010在表t_fxcw_bm_fxjs中取出的XMCD字段值（dc）的长度等于4，而不是等于2
> * 这样会出在在存储过程中的判断 UPPER(T.XMCD)='DC' 会查不出记录，所以不会跑p2010的数据
> * 解决方法：update t_fxcw_bm_fxjs set xmcd='dc' where zbxmbm='p2010'
> * 根本解决方法是在存储过程中把 UPPER(T.XMCD)='DC' 去除其他字符，包括空格
